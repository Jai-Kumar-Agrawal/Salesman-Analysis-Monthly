## 📊 Salesman Performance Analysis Dashboard (Excel + VBA)

### 🔹 Project Overview

* Built an interactive dashboard to analyze monthly performance of sales representatives across multiple cities
* Designed using Excel with Pivot Tables, Charts, and VBA automation
* Consolidated all insights into a single, user-friendly dashboard for better decision-making

### 🔹 Key KPIs

* Total Sales: 38,945
* Average Sales: 276.20
* Average Target Achievement: 55.24%
* Average Distance from Target: 44.76%

### 🔹 Dashboard Features

* Created 4 Pivot Tables for:

  * Top Salesman (Sales-wise)
  * Target Hit %
  * Away from Target %
  * Sales Distribution Analysis
* Visualizations used:

  * Bar Chart (Top Performers)
  * Line Chart (Target Hit Trend)
  * Pie Chart (Away from Target %)
  * Donut Chart (Sales Distribution)

### 🔹 Interactivity & Automation

* Added City-wise slicer for dynamic filtering
* Implemented VBA-enabled checkboxes to:

  * Control specific Pivot Tables independently
  * Toggle selected dashboards dynamically
